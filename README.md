# 개요

- 주수모니터링 시스템 frontend
- 대시보드, 카메라 모니터링 뷰어

# 전체 구성환경

# 안녕하세요

- frontend : vue3
- backend : springboot

## 적용되는 기본 환경

- Vue Framework: `vite`, `vue3`, `pinia`, `vue-router`
- i18n: vue-i18n
- Style: SASS, SCSS, vuetify
- Rest Communication: Axios
- Chart: highchart, echart
- Grid: tuigrid, tabulator
- 주요 라이브러리 추가 예정
